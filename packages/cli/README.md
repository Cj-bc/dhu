# @dhu/cli

```
dhu v0.0.18

Usage:
  $ dhu <command> [options]

Commands:
  login      Save login info to local data path
  logout     Remove login info from local data path
  gpa        Get GPA
  atte       Get attendance
  info       Get info
  task       Get tasks
  timetable  Download timetable csv

For more info, run any command with the `--help` flag:
  $ dhu login --help
  $ dhu logout --help
  $ dhu gpa --help
  $ dhu atte --help
  $ dhu info --help
  $ dhu task --help
  $ dhu timetable --help

Options:
  -h, --help     Display this message
  -v, --version  Display version number
```
